Cloudflare + Subdomain (nina.efeservili.dev)

  1. Cloudflare Dashboard → Workers & Pages → Create application → Pages.
  2. Bu GitHub reposunu bağla, build komutu boş bırak, output directory: website.
  3. Deploy bitince Pages projesinde Custom domains → Set up a custom domain.
  4. nina.efeservili.dev ekle ve doğrula.
  5. DNS otomatik yönetilmiyorsa Cloudflare DNS’te CNAME kaydını Pages hedefiyle eşle.
deploy-checklist 
[ ] GitHub repo connected to Cloudflare Pages
[ ] Build command empty
[ ] Output dir = website
[ ] Custom domain = nina.efeservili.dev
[ ] HTTPS active
